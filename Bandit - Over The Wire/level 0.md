You need to connect with bandit's server by the way 

`ssh bandit<level>@bandit.labs.overtherwire.org -p 2220`   . Ex: ssh bandit0@bandit.labs.overtherwire.org -p 2220  . At level 0 : pass is bandit0

And you need to solve each level to find the password for the next level.

OverTheWire is a game that help you can learn about basic commands and some fundamental knowledges.
